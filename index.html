☁ 클라우드 동기화 미연결

구글시트에 자동 저장되어 어느 기기·브라우저에서 열어도 같은 데이터가 나옵니다. 아래 [설정 방법 보기]를 먼저 따라 하세요.
변경할 때마다 자동 저장
최초 1회 설정 (약 10분, 한 번만)
  1. 구글드라이브(drive.google.com)에서 새 구글 시트를 만듭니다. (이름 예: 학생관리 데이터)
  2. 시트 상단 메뉴 확장 프로그램 → Apps Script 클릭.
  3. 편집기의 기본 코드를 모두 지우고, 아래 [코드 복사]로 복사해 붙여넣기 후 저장(💾).
  4. (암호를 쓰려면) 코드 2번째 줄 TOKEN=''의 따옴표 안에 원하는 암호를 넣습니다.
  5. 오른쪽 위 배포 → 새 배포 → 톱니바퀴에서 유형 웹 앱 선택.
  6. 실행 대상 나, 액세스 권한 모든 사용자 → 배포.
  7. 권한 요청이 뜨면 본인 구글 계정으로 허용.
  8. 표시된 웹 앱 URL(.../exec)을 복사해 위 웹앱 URL 칸에 붙여넣고, 암호를 썼다면 암호도 입력.
  9. 연결 저장·테스트 → 성공하면 ⬆ 클라우드에 올리기로 지금 데이터를 올립니다.
  10. 다른 기기에서는 이 파일을 열고 같은 URL(+암호) 입력 후 ⬇ 클라우드에서 불러오기. 이후엔 자동 동기화됩니다.
⚠ 두 기기에서 동시에 편집하면 나중에 저장한 쪽이 덮어씁니다. 한 번에 한 기기에서만 편집하세요.
